Qué significa

Contexto

La movilidad social se refiere al cambio en la condición socioeconómica de las personas. El CEEY la relaciona con igualdad de oportunidades: cuando las circunstancias de origen pesan menos sobre los logros, la movilidad social es mayor.

Usar la API

POST /v1/predict

Para navegador, el backend debe permitir CORS desde tu dominio de GitHub Pages. El frontend usa credentials: "omit" y sólo envía Content-Type: application/json.

Nota ética. Este demo traduce variables del hogar en una predicción técnica. No evalúa el valor, potencial ni mérito de una persona. Úsalo para explicar modelos, APIs y experimentación responsable.
